Chronicles Online Item Manager

Chronicles Online is a fictitious RPG that I wrote an item manager for as a practice project.



It features loading item data from json, editing it, and saving it back to json, supporting different item types.
This was mainly used as a practice project to learn WPF and MVVM, following MVVM principles as strictly as possible.
The project is built in an extensible and async fashion.
